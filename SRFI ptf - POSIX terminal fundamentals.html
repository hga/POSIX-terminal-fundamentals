###: POSIX Terminal Fundamentals

by John Cowan (architect and author) and Harold Ancell (author and editor)

Abstract

A set of procedures to support command line and terminal interface programs, and serial interfaces to embedded and the like hardware.

POSIX provides a complete set of routines for manipulating terminal devices — putting them in "raw" mode, changing and querying their special characters, modifying their I/O speeds, and so forth. However, now that terminal emulators have almost completely displaced terminals, very little of this is useful except for directly controlling serial-line hardware, which is still widely used for embedded programming. This SRFI provides useful serial line procedures, and high level safe wrapper for use by command-line programs. Once you go to the trouble of implementing the with-* and without-echo procedures, including serial terminal control ones is not much extra work given Chibi Scheme sample implementation provided.
